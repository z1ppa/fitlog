Продукт

FitLog — прогрессивное веб-приложение (PWA) для трекинга тренировок и анализа фитнес-прогресса. Цифровой дневник спортсмена с встроенной аналитикой, библиотекой программ тренировок и отслеживанием антропометрии.

ЦЕЛЕВАЯ АУДИТОРИЯ
  • 👤 Любители фитнеса и посетители спортзалов
  • 🏅 Спортсмены со структурными программами
  • 📊 Пользователи, следящие за прогрессом
  • 🛡️ Тренеры (через админ-панель)
КЛЮЧЕВЫЕ ЦЕННОСТИ
  • 📱 Mobile-first, работает офлайн
  • 🏋️ Методика Сарычева (силовые программы)
  • 📈 Кастомная аналитика без сторонних зависимостей
  • 🔒 Ролевая система доступа (RBAC)

Функциональность
💪

Тренировки

Логирование упражнений, подходов, весов. Старт по шаблону программы. Batch-создание за один запрос.

📋

Программы тренировок

Публичные и личные программы. Методика Сарычева — расчёт весов из 1ПМ по схеме «4×2(70%)».

📊

Аналитика

Стрики, объём нагрузки, недельная активность, топ-упражнения. Кастомные SVG-графики.

📏

Антропометрия

11 параметров: вес, обхваты, % жира. История измерений с линейным графиком.

🏋️

Библиотека упражнений

Поиск с учётом ё/е. 9 групп мышц. Кастомные упражнения от пользователя.

🛡️

Админ-панель

Управление пользователями. Метрики платформы: DAU, активность, топ-упражнения за 30 дней.

📱

PWA / Offline

Кастомный Service Worker. Кэш ключевых страниц. Установка на главный экран телефона.

🔐

Безопасность

JWT в httpOnly cookies. bcryptjs для паролей. Серверная проверка сессии на каждом endpoint.

🗄️

Миграции

Prisma Migrations, 8 версий схемы. Контролируемая эволюция базы данных.

Итоговая оценка

✅ Сильные стороны

  • ✅ Современный стек: Next.js 16, React 19, Prisma 7
  • ✅ Mobile-first PWA с поддержкой офлайн
  • ✅ TypeScript strict mode — высокая надёжность кода
  • ✅ Готовая RBAC-система (USER / ADMIN)
  • ✅ Кастомная аналитика без лишних зависимостей
  • ✅ Vercel + Supabase — быстрый старт и деплой
