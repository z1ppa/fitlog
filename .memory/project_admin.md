Добавлена полноценная админ-панель.

**Схема:** добавлен `enum Role { USER ADMIN }` и поле `role` в модель User (миграция применена).

**API роуты:**
- `GET /api/admin/users` — список всех юзеров со статистикой
- `GET/PATCH/DELETE /api/admin/users/[id]` — детали юзера, смена роли, удаление
- `GET /api/admin/analytics` — агрегированная аналитика

**Страницы:**
- `/admin` — список юзеров с поиском, сменой роли, удалением
- `/admin/users/[id]` — детальная страница юзера (тренировки, замеры, статистика)
- `/admin/analytics` — дашборд с 4 бар-чартами (DAU, тренировки/день, объём, регистрации), топ упражнений, распределение активности юзеров

**Доступ:** только пользователи с `role === "ADMIN"`. Кнопка "⚙️ Админка" появляется на дашборде только у админов.

**Первый админ:** `kulik.inc@gmail.com` назначен ADMIN через скрипт.

**Why:** Запрос пользователя для контроля всех зарегистрированных юзеров.
**How to apply:** При добавлении новых моделей данных — добавлять их в admin API и страницы.
