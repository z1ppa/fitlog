FitLog — приложение для трекинга тренировок. Next.js 16 + TypeScript + PostgreSQL (Neon) + Prisma 7 + NextAuth + Tailwind.

Реализовано: авторизация, тренировки с таймером и подходами, библиотека упражнений, замеры тела, профиль с BMI, дашборд. Тёмная тема, мобильный дизайн, весь UI на русском.

**Why:** Личный проект пользователя для трекинга своих тренировок.
**How to apply:** Не ломать существующий стиль и паттерны при добавлении новых фич.
